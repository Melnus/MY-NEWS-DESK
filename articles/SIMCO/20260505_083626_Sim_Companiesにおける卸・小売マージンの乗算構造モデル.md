# TITLE: Sim Companiesにおける卸・小売マージンの乗算構造モデル

# ABSTRACT
Sim Companiesの価格形成を、単なる市場価格ではなく「流通段階ごとの利益分離」を目的とした原価×マークアップの積構造として定義する。生産・卸・小売の各レイヤーは独立したマージンを持ち、サプライチェーン全体での最適化問題として価格が決定される。

# DETAIL

### 1. 基本数理モデル（構造価格の定義）
最終販売価格 $P$ は、生産コスト $C$ に対する卸率 $w$ と小売率 $r$ によって決定される。

> **P(C, w, r) = C · (1 + w) · (1 + r)**

※ w, r は市場均衡・在庫・需要制約に依存する内生変数である。

---

### 2. 利益の階層分解

* **生産者利益（Upstream）**

C · w

* **卸業者利益（Midstream）**

C · (1 + w) · r

* **小売業者利益（Downstream）**

C · (1 + w) · r

（※ 小売は卸価格との差分として定義される）

---

### 3. レバレッジ構造（偏微分）

価格関数の偏微分：

∂P/∂w = C(1 + r) ∂P/∂r = C(1 + w)

→ 一方のマージン増加は他方の限界利益レバレッジを増幅する

---

### 4. 管理会計的視点

本構造は「価値ベース価格」ではなく「マークアップ連鎖構造」である。

* 卸 = Upstream Margin
* 小売 = Downstream Margin

価格はサプライチェーンの積構造として形成される。

---

# CONCLUSION

1. Sim Companiesの価格は P = C(1+w)(1+r) の積構造で支配される  
2. 利益はレイヤーごとに完全分解可能である  
3. 最適化問題は単一利率ではなく (1+w)×r の相互作用最大化である  
4. 本モデルにより流通構造のレバレッジ源泉を定量的に特定できる
