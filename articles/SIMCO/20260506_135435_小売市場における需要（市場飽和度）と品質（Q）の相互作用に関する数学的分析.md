# TITLE: 小売市場における需要（市場飽和度）と品質（Q）の相互作用に関する数学的分析

# ABSTRACT 
本稿では、Sim Companiesにおける小売販売において、商品の「需要」と「品質（Q）」が販売速度および時間あたり利益（時給利益）に与える影響について分析する。
公式記事にて公開された小売方程式（Retail Equation）と実測データに基づき、品質向上がもたらす効果が商品の単価および市場の飽和状態によって劇的に異なる理由を解明し、リサーチ投資における合理的な経営判断の基準を提示する。

# DETAIL
## 1. 市場飽和度（Saturation）の正体と構造的ペナルティ
百科事典に表示される「需要（とても低い～極めて高い）」は、内部パラメータである「市場飽和度（S）」によって決定される。このS値は「サーバー全体のプレイヤーが過去24時間に小売で販売した総量」に依存する。
特に食品などのチュートリアル産業は、新規参入者から放置プレイヤーまで生産・販売のハードルが極めて低いため、構造的に常に「超・供給過多（需要：低い～とても低い）」に陥り、販売方程式において強力なペナルティ（ブレーキ）が常時かかっている状態となる。

## 2. 品質（Q）の実効メカニズム
公式の小売モデルにおいて、品質（Q）の向上は「販売速度の単純な倍率バフ」としては計算されない。数式上、品質は**「市場飽和度（S）を直接減算する（実効飽和度を引き下げる）」**パラメータとして機能する。
> 公式仕様：品質が1上がるごとに、実効飽和度を0.24下げる（最大 -0.38まで）

## 3. レバレッジ効果の格差（高単価 vs 低単価）
上記の方程式構造により、商品の特性によって品質投資へのリターン（ROI）に決定的な格差が生じる。

*   **高単価・低飽和商品（トラック、電気自動車など）**
    参入障壁が高く元の飽和度（S）が低いため、QによるSの減算効果が方程式の分母を劇的に縮小させる。結果として、Q0からQ6へ向上させた際、時間あたりの総利益が数千ドル単位で爆発的（非線形的）に増加する。
*   **低単価・高飽和商品（食品、スニーカーなど）**
    元の飽和度（S）が過大であるため、Qを限界まで上げてSを減算しても、方程式全体への影響が微小（焼け石に水）に留まる。実測データにおいても、Q0からQ6へ向上させても時間あたり利益は数十ドル程度しか増加せず、増分が店舗の賃金コスト（AO）を十分にカバーできない「品質の罠」が存在する。

# CONCLUSION
小売市場において「品質を上げれば売上速度が比例して伸びる」という認識は、計算アルゴリズム上誤りである。
品質（Q）は「需要の悪さ（飽和度）を削り取る係数」であり、元の需要と単価という土台が小さければ、その恩恵は数学的に打ち消される。
したがって、莫大なコストを伴うリサーチ投資（品質の向上）は、「単価が高く、かつ需要が『高い』以上の中～上位産業」に限定して実行すべきである。低単価・高飽和産業において利益率が低下した場合、品質向上による打開を図るのではなく、他産業へのピボット（業態転換）、あるいは生産・卸売特化への移行が合理的な経営戦略となる。
