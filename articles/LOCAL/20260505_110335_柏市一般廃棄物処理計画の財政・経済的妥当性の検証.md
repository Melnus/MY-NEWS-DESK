# TITLE: 柏市一般廃棄物処理計画の財政・経済的妥当性の検証

# ABSTRACT 
本セッションでは、柏市が策定した「一般廃棄物処理基本計画（令和5年度）」および「清掃事業概要（令和6年度）」を基軸に、同市のインフラ更新計画の妥当性を多角的に分析した。主要な知見として、(1)インフレ率を遥かに上回る予算の急膨張、(2)全国平均を大きく逸脱する市民1人あたりの負担額、(3)稼ぐ力（商業吸引力）の減退と固定費（ハコモノ維持費）増大の矛盾が浮き彫りとなった。行政側が主張する「インフラ維持の不可避性」に対し、財政の硬直化と将来世代への負担転嫁というリスクが統計データによって裏付けられた。

# DETAIL

### 1. ゴミ処理コストの異常値とインフレ乖離
柏市の清掃事業予算は、令和5年度の約61.2億円から令和7年度の約140.5億円へと、**わずか2年間で約2.3倍（+130%）**に急増している。
*   **1人あたり負担額の比較:** 令和7年度予算ベースで市民1人あたり**約32,000円**。これは環境省統計（令和5年度調査）による全国平均**18,300円の約1.75倍**に達し、明確な「異常値」を示している。
*   **インフレ要因の検証:** 建築費指数や消費者物価指数の上昇（同期間で約10〜15%）ではこの130%の予算増を説明できず、特定の施設（北部クリーンセンター等）への投資集中と単独維持方針がコスト増の主因であると特定された。

### 2. 財政の硬直化と自立性の欠如
「第6編 財政」の指標により、柏市の家計簿が極めて脆弱な状態にあることが確認された。
*   **経常収支比率 92.7%:** 歳入の約93%が人件費、扶助費、公債費等の固定費で占められており、政策的に使える余力が7%しかない「硬直化した身体」の状態にある。
*   **財政力指数 0.91:** 人口増の恩恵を受けつつも、依然として自力で標準的な行政サービスを賄えず、国からの「仕送り（地方交付税等）」に依存する構造が続いている。

### 3. 地域経済の衰退とハコモノ投資の逆行
「第14編 経済産業」の統計により、市の再開発計画と市場動向の乖離が判明した。
*   **商圏吸引率の低下:** 平成28年度の29.6%から令和3年度の24.0%へ急落。近隣市（流山市等）への顧客流出が顕著である。
*   **民間企業の撤退:** 令和6〜7年にかけて駅前大手商業施設が相次いで閉店。プロの経営者が「収益性なし」と判断した地域に対し、行政が巨額の公金を投じてハコモノ（東口再開発）を維持・更新しようとする「顧客不在の経営」が示唆された。

### 4. 人口動態の変化による税収の質の劣化
推計人口（常住人口）の精査により、頭数としての人口は維持されているものの、その中身が変質している。
*   **世帯の小規模化:** 1世帯あたり人員が2.08人まで減少。単身者や外国人住民が増加傾向にある。
*   **逆ザヤリスク:** 1人あたり3.2万円の処理コストに対し、控除の大きい子育て世帯や低所得層の転入が増えることで、既存の定住市民（納税者）1人あたりの実質的な肩代わり負担が増大する構造的欠陥がある。

# CONCLUSION
柏市の廃棄物処理および都市開発計画は、「インフラ維持＝必須」という行政の論理を優先するあまり、**「インフレ率を超越したコスト膨張」**と**「市民1人あたりの負担額の異常性（全国平均の約2倍）」**を看過している。
基本計画において「コスト（円）の目標値」が欠落していることが、実施段階での予算暴走を許すガバナンス不全を招いている。持続可能な都市経営のためには、単独維持の固執を捨てた「広域連携による施設の徹底集約」および「ゴミ袋有料化による受益者負担の適正化」といった、痛みを伴うダウンサイジング（縮小均衡）への転換が急務である。
